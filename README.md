# Privacy Policies

This repository hosts privacy policies for various applications and services, accessible via GitHub Pages.

## Available Privacy Policies

- [Flippy Boi](flippy-boi/privacy-policy.html)

## GitHub Pages Setup

This repository is configured to work with GitHub Pages. To enable GitHub Pages:

1. Go to your repository on GitHub
2. Click on **Settings** (in the repository navigation bar)
3. Scroll down to the **Pages** section (in the left sidebar)
4. Under **Source**, select:
   - **Branch**: `main` (or `master` if that's your default branch)
   - **Folder**: `/ (root)`
5. Click **Save**

After enabling, your privacy policies will be available at:
- `https://folterung.github.io/privacy-policies/`
- `https://folterung.github.io/privacy-policies/flippy-boi/privacy-policy.html`

**Note**: It may take a few minutes for GitHub Pages to build and deploy your site after the first setup.

## Adding New Privacy Policies

To add a new privacy policy:

1. Create a new folder for your application/service (e.g., `my-app/`)
2. Add your `privacy-policy.html` file inside that folder
3. Update `index.html` to include a link to your new policy
4. Commit and push your changes

The new policy will automatically be available via GitHub Pages after deployment.
